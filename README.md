## Bright-TV Introduction
BrightTV is a subscription-based video streaming platform that provides digital content to its users. The company is focused on expanding its customer base and increasing user engagement through data-driven decision-making.


- For the current financial year, the CEO has set a strategic objective to grow the platform’s subscription base. To support this goal, the Customer Value Management team has been tasked with analyzing user behavior and identifying opportunities to improve both acquisition and retention.<br>- In this project I will be analysing userprofiles and the viewership trends of Bright TV as a junior data analyst and giving insights based on the outcomes.

## Project planning description and raw data
As a junior data analyst I was given two sets of data, being the viewership and user profiles. The raw data was in CSV and I had to separate the tables individually before I can analyze them.This task was done in Microsoft Excel.<br>The dataset provided for this case study includes:


   1. User profile information (demographics)
   2. Viewer transaction data, where each record represents a single viewing session

## Data Processing
- After the datasets were separated in Microsoft Excel, they are exported to Databricks for processing. In Databrick, we used data ingestion where the tables are stored in the workspace catalogue and schema.
- Here we use SQL to answer queries and get insights on the data. In this case of two tables, SQL JOINS are used to combine the tables into one and CASE statements to add other columns.


KPIs include:


- Subscription Growth Rate: Tracks increase in total subscribers (primary business goal)
- Active Users: Measures platform usage and engagement levels.
- User Retention & Churn Rate: Evaluates how well users are retained over time.
- Total Viewing Sessions: Indicates overall platform activity (per-session dataset).
- Average Watch Time & Sessions per User: Measures depth of engagement.
- Peak vs Low Consumption Trends: Identifies usage patterns across days/times.
- Top Content Performance: Highlights content driving the most engagement.
- User Acquisition Rate: Tracks growth of new users


These KPIs collectively help identify drivers of consumption, engagement gaps, and opportunities to grow the user base.
